How the waterfall builder works

  1. 01Sign in ✓
  2. 02Sync your AdMob apps & ad units → Apps
  3. 03Open the Builder → Builder
  4. 04Select an app + one or more source ad units
  5. 05Choose country targeting (Global / Choose / Exclude)
  6. 06Set waterfall depth (1–{{ max_lines }}) + floor type
  7. 07Fetch live AdMob report (last 7 days)
  8. 08Review/edit the calculated tier eCPM values
  9. 09Push to AdMob: For each source ad unit: (a) create N labeled tier ad units, (b) replicate every bidding mapping (Meta/AppLovin/etc.) from the source onto every tier, (c) create ONE mediation group targeting source + all tiers, with N MANUAL AdMob Network waterfall lines + AdMob Network LIVE bidding line (plus a LIVE line per replicated 3P network).
  10. 10Tool reads the group back and audits every line's state. Disabled lines are surfaced as errors.
